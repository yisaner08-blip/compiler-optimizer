#### 1.7.1 不变量外提

如果循环中的一个计算在循环的每次迭代都产生相同的值，则该计算被称作循环不变量。

当循环的基本块的一个操作的操作数都满足下列条件时，该操作是循环不变的：

1. 操作数是常数
2. 所有到达操作数的定值都在循环之外
3. 只存在一个到达操作数的定值，并且该定值在循环内，并且该定值本身是循环不变的

标记循环中的不变量集合： 1.首先标记两种简单的不变量：标记所有操作数都是常数的指令为不变的；标记所有操作数的到达定值都在循环外的指令为不变的。
2、循环直到找出了所有的循环不变量：
标记操作数符合下面条件，并且没有被标记的指令，为不变的：操作数是循环不变量，或者只有一个定值到达操作数，并且该定值已经标记为循环的不变量。

不变量外提条件一：语句 s 是循环 L 所有出口的必经节点。
不变量外提条件二：循环中对 x 的所有使用只能被 s 中的定值到达。
不变量外提条件三：v 不能在循环 L 内有其他定值。
不变量外提条件四: 不变量的使用的 x、y 的定值都来自于循环之外。

算法示例：
执行到达定值分析，并构建 UD 链
找出循环 L 中的循环不变量
对每个循环不变量指令 s，假设 s 定值 v，使用 x 和 y，执行{
如果 （满足不变量外提条件一&&不变量外提条件二&&不变量外提条件三&&不变量外提条件四）
{将 s 外提，移到 L 的前置节点}
}

如果循环迭代次数为 0，就由先测试循环是否执行来避免循环体的执行（重要）

#### 1.7.2 归纳变量和强度削弱

归纳变量:在循环中，如果变量 x 的值每次改变都是增加或者减少某个固定的常数，那么 x 就是循环的归纳变量

基本归纳变量：只有唯一的形如 i:=i±C 的定值，C 是常数
导出归纳变量：变量 j 在循环中的定值总是可化归为归纳变量 i 的同一线性函数，即 j=C1 异或 i±C2，其中 C1 和 C2 都是循环不变量

归纳变量算法示例：
找出循环 L 中的循环不变量
IV=空集
//识别基础归纳变量
对循环 L 中每一条语句 S，循环
如果（S 形如 i ⬅ i±C 并且 C 是循环不变量）{
IV = IV∪{i}； //i 是基本归纳变量
与 i 关联的三元组为(i,1,0)；
}
//识别导出归纳变量
如果（变量 j 在循环 L 中只有一次定值，并且定值语句 S 具有以下形式之一：
j = C*k，j=k*C，j=i/C，j=k±C，j=C±k，其中 k 是归纳变量，C 是循环不变量）{
如果（k 是基本归纳变量）{
IV = IV∪{j}；
根据 S，计算与 j 关联的的三元组：
、case j = C*k 或者 j=k*C: 三元组为(k,C,0)
case j =k±C: 三元组为(k,1,±C)
case j=k/C： 三元组为(k,1/C,0)
case j=C+k： 三元组为(k,1,C)
case j=C-k： 三元组为(k,-1,C)
}
否则{
假设 k 属于 i 族（i 是基本归纳变量）
如果（循环 L 中对 j 的定值和对 k 的定值之间没有修改 i，并且 循环 L 外没有 k 的定值到达 S）{
IV = IV{j}；
根据 S 和与 k 关联的三元式（i，C1，C2）计算与 j 关联的三元式
}
}
}

强度削弱算法示例：
依次处理每一个基本归纳变量 i，对每一个与 i 同族的归纳变量 j，其三元式为(i,C1,C2)，执行以下步骤：

1. 建立新变量 s，如果归纳变量 j1 和 j2 三元式相同，则只创建一个新变量
2. 用 j=s 代替原来对 j 的赋值
3. 在 L 中，紧跟在 i=i+C（C 是循环不变量）语句之后，插入语句
   s = s + C1*C
   如果 C1*C 是常数，则计算常数值；C1*C 是循环不变量，则引入临时变量 t = C1*C，s = s+t；
   通过不变量外提优化，外提到循环前置节点；
4. 在前置节点末尾为 s 设置初值 s=C1\*i+C2
5. 用 s 替换源程序中每一个 j

#### 1.7.3 循环展开

只进行简单循环变换，不改变迭代执行顺序

分为头尾展开和体展开
头尾展开：头尾展开指的是将循环的首迭代或者尾迭代删除，形成单独的代码，也被称为循环剥除（Loop Peeling）；删除和循环控制变量相关的条件分支语句，简化循环体。
体展开：扩大循环体，为公用子表达式删除、强度削弱、指令调度等提供更多的机会，提高数据局部性

循环展开优化框架：

1. 根据循环展开代价模型，判断循环展开收益，选择需要展开的循环
2. 选择合适的展开因子 u
3. 将循环体复制 u 份，调整相应的循环索引变量
4. 形成和处理尾循环
   第 1 和 2 步是循环展开的关键，考虑：
5. 目标机的体系结构,例如，硬件寄存器的个数，指令 Cache 的大小和组织结构，以及指令流水线等
6. 与循环本身有关，包括循环体的大小、操作数的类型和个数
